# 御守社后续构建手册

本文是后续 AI 接手项目时的唯一核心文档。请先读完本文，再查看代码。本文只保留对后续构建有帮助的信息。

## 一、项目定位

`御守社` 是一个东方 Project 同人风格的抽签网页。目标不是普通抽签按钮，而是“地点选择 + 场景化抽签 + 结果签面 + 收藏记录”的沉浸式网页。

视觉方向：

- 明亮、精致、日式二次元、视觉小说式场景。
- 地点必须有清晰辨识度，不允许只换色。
- 主视觉必须是完整场景图，不使用真实照片作为运行页面背景。
- 不允许回退到简单线条、几何图形、抽象图标堆叠来假装场景。
- 现在的主要短板不是功能骨架，而是“东方味的颗粒度”还不够。后续改动要让场景更像幻想乡里的日常异变现场，而不是泛用高精度日式幻想壁纸。
- 场景不只是背景。每个地点都应明确自己的道具、声音、纸张、风向，以及一处轻微不合常理的小细节。
- 更新素材时不要追求复杂纹理或密集装饰。优先保持与现有 `generated-themes` 场景一致的干净动漫背景风格，只用少量高辨识度道具校正地点气质。

路线差异要求：

- 博丽神社是参拜式：铃绳、赛钱箱、旧木匣、御札、结界回声，节奏应直白、利落、带一点懒散的正统神社味。
- 红魔馆是茶会式：银托盘、红茶、玫瑰、窗纱、怀表或餐具轻响，节奏应更优雅、更任性，不要只做成红色神社。
- 永远亭是问药式：药签匣、药包、月兔脚步、竹影、纸门，节奏应像问诊和配药，建议可以温柔但要清楚。
- 后续新增路线也必须先定义“仪式类型”，再补图、文案和动效（注意生成素材时不使用复杂纹理，与现有风格一致即可）。

签文方向：

- 签文库要逐步去模板化。现在很多句式稳定但偏现代心灵鸡汤，后续要更像角色本人在说话。
- 避免批量复用“今天借用某角色气质提醒你”“不用急着证明什么，先把真正重要的一步走稳”等结构。
- 每条签文应至少有一个角色专属动作、地点物件或口吻拐点，让读者能分辨是谁递出的签。

设定依据：

- 设计地点、角色、签面、抽签道具前，优先查 `https://thbwiki.cc/`。
- 可参考 Touhou Wiki、萌娘百科、官方或官方授权资料进行交叉确认。
- 当前项目仅本地使用，不考虑公开传播授权流程

## 二、当前实现基线

正式路线目前已经扩展为 14 处：

- `hakurei`：博丽神社
- `scarlet`：红魔馆
- `eientei`：永远亭

- `hakugyokurou`：白玉楼
- `forest-of-magic`：魔法森林
- `myouren-temple`：命莲寺
- `palace-of-earth-spirits`：地灵殿
- `sanzu-river`：三途川
- `misty-lake`：雾之湖
- `human-village`：人间之里
- `sunflower-field`：向日葵田
- `divine-spirit-mausoleum`：神灵庙
- `nameless-hill`：无名之丘

关键数据结构：

- `src/types/omikuji.ts`
  - `OmamoriRouteId` 覆盖 14 处正式签路。
  - `OmamoriRouteConfig`
  - `Fortune`
  - `DailyDrawRecord`
  - `SavedFortune`
- `src/constants/fortune.ts`
  - `OMAMORI_ROUTES`
  - `OMAMORI_ROUTE_LIST`
  - `SCENE_CANDIDATES`
  - `DEFAULT_SETTINGS`
  - `DRAW_TIMINGS`
- `content/omikuji/fortunes.json`
  - 签文数据源。
  - 每条签文通过 `relatedSceneIds` 标记相关场景；正式抽签会按当前 `OmamoriRouteId` 过滤签池，不再从全池抽取。

关键 UI 模块：

- `src/components/daily-draw-panel.tsx`
  - 当前入口选择和抽签主流程在这里。
  - `RouteSelectionPanel` 已能显示 14 处正式场景，进入后会使用对应路线的背景、仪式拆层、音效和签池。
  - 当前主页底部状态文案已统一为 `14 处正式签路已开放`。
- `src/components/shrine-scene.tsx`
  - 抽签场景外壳、背景图、前景叠层、全局提示。
- `src/components/fortune-card.tsx`
  - 结果签面展示。
- `src/components/bell.tsx`
  - 仪式铃动画。
- `src/components/fortune-tube.tsx`
  - 签筒 / 托盘 / 药签匣动画。
- `src/components/collection-list.tsx`
  - 签册页，按路线过滤签册。
- `src/components/settings-modal.tsx`
  - 音频、动效、默认入口等设置。
- `src/store/omamori-store.ts`
  - 当前路线、今日抽签、本地收藏和设置状态。
